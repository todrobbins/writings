# Like wind tossed global
_May 2006_
```
Like wind tossed global
Like water rained down
Like light burning
Like a city on a hill
For the ages spoke of knowledge spread like wildfire
Like eternal breadth
Like flooding rays
Peace from our Lord
Intelligence for generations
```
